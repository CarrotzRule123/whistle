## CLI

### Installing

```base
$ git clone https://github.com/whistle-lang/whistle.git
$ cd /whistle/cli
$ cargo build
$ export PATH=$PATH:/place/with/the/file
```

### Usage

```
$ whistle -h
```

### Example

```
$ whistle tokenize <input file> <output file>

```
